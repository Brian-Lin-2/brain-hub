### Common Elements

`Client` - User or desktop icon
`Load Balancer` - Rectangle with split arrows
`Server/API` - Rectangle
`Database` - Cylinder
`Cache` - Horizontal Rectangle
`Decision Box` - Diamond
`Sub-Process` - Rectangle with double vertical lines on the sides
