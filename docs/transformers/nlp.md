# NLP

Natural Language Processing is a branch of AI that gives computers the ability to understand, interpret, and generate human language.

## Reference

[Slides](https://cme295.stanford.edu/slides/fall25-cme295-lecture1.pdf)

## Main Types

### Classification

Takes in an input, spits out a single label

**Common Types:**

- `Sentiment Extraction` - Label is generally positive or negative (emotions)

### Multi-Classification

Takes in an input, spits out multiple labels

**Common Types:**

- `Named Entity Recognition` - Within a sentence, the LLM is able to determine which word is the entity

### Generation

Takes in an input, spits out an output

**Common Types:**

- `Machine Translation` - Given a sentence, the LLM is able to spit out a translation of it
